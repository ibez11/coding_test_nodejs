## Installation

```bash
git clone https://github.com/ibez11/coding_test_nodejs
cd coding_test_nodejs
npm install
```

Then,
```bash
composer require ajthinking/php-file-manipulator @dev
php artisan vendor:publish --provider="PHPFileManipulator\ServiceProvider"
```
